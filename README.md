# Camera Slider
DIY camera motorized rail - To get great timelapse in movement!

In development, soon.

## Shopping list
* 1x Arduino Uno (Rev3)
* 1x Rectifier Diode (1N4001)
* 1x LCD screen (16x2)
* 1x DC Motor
* 1x NPN-Transistor
* 1x Trimmer Potentiometer 10kΩ
* 3x 10kΩ Resistor bands 4; tolérance ±5%
* 1x 220Ω Resistor bands 4; tolérance ±5%
* 3x Pushbutton
